권한 추가 및 관리

📲게시글나가기

1. 로그인 후 최근 알림 확인

2. 내 게시글 및 권한 관리 클릭

3. 닉네임에 게시글에 영역 작성 권한을 주고 싶은 사용자의 닉네임을 적고 권한추가를 클릭

4. 권한이 추가되면 해당 게시글은 협동 게시글이 되며, 권한을 가진 사람들 목록이 밑에 보인다

5. 영역 추가 권한을 얻은 사용자는 영역을 추가 할 수 있다

단, 영역 위치를 수정하고 싶은 경우는 해당 영역의 작성자만이 수정할 수 있다.
또한 해당 영역의 내용도 해당 영역 생성자만이 수정 및 삭제를 할 수 있다.
(단, 게시글의 주인은 모든 권한을 가지고 있다)
(빨간색 : 다른 사람이 작성한 영역 -> 공개)
(노랑색 : 다른 사람이 작성한 영역 -> 비공개)
(보라색 : 내가 작성한 영역)
